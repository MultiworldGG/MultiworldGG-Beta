# Into the Breach

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export a config file.

## Goal

- Beat the game with any squad after getting the right amount of achievements

## Locations

- Achievements (only the squad achievements)
- Clearing 1, 2, 3 and 4 islands

## Items

- Starting defense and grid power (all the defense and grid power is taken away to be put into the item pool when starting the multiworld)
- Starting power cores
- Squad unlocks (Gathering X coins items to get a single squad wasn't interesting and wouldn't work with the logic anyway)
- Temporary boosts
- Traps

## Features

- Squad randomization
- Receiving gifts from other worlds
- Play only with the custom squad
- Difficulty requirements to achievements/beating the game can be set on the yaml
- Energylink of reputation in the shop

## Planned features

- Make islands unlock items
- Add custom achievements
