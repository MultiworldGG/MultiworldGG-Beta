# Sonic Heroes for Multiworld.gg

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export a config file.

## How does Sonic Heroes work with AP?
The entire game is handled through the level select (challenge) menu and separates the game into gates. Each gate has a random boss that can be unlocked by collecting a specified number of emblems and that must be beaten to unlock the next gate. Emblems among many other items can be collected by completing stages, killing enemies with Team Dark, collecting rings with Team Rose, and completing objectives with Team Chaotix.

The final gate will always have Metal Madness as a boss. Your game ends on beating Metal Overlord.

## FAQ:
### I can't open the Config window after updating

I changed the PlaySounds option to a boolean and reloaded cant handle having the old default value.
Go to {Reloaded Directory}/User/Mods/sonicheroes.archipelago.client and delete the Config.json
Relaunch Reloaded and it will work.

### I like the Old Version More

If there is enough support, I will port the old version over (though it will NOT have ability and character lockout)

### Why is Triangle Jump/Flying yellow in the UI?

Triangle Jump requires Homing Attack and Flying requires Thundershoot

### Music Rando is on but only vanilla music is playing

If you have followed the directions correctly then make sure that your file paths are not too long (aim for less than 100 characters)
If your pool of music is smaller than vanilla, then vanilla is forced as a failsafe
