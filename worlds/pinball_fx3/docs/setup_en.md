# Pinball FX3 Randomizer Setup Guide

## Prerequisites 

- If you do not use MWGG: Install the Pinball FX3 APWorld as you would any other custom APWorld
- Have a multiworld generated and a room hosted

## How to play

- Open Pinball FX3
- Open the MultiworldGG Launcher. Find and open the Pinball FX3 Client
- Connect to the multiworld room with the server credentials, e.g. multiworld.gg:50125
- Send the `/pinball` command
- You are ready to play! Make sure to check out the Pinball FX3 tab in the client
