# Setup Guide for Archipela-Go!

# Required Software
* [MultiworldGG](https://github.com/MultiworldGG/MultiworldGG/releases)
* [Latest Android Mobile App (.apk)](https://github.com/aki665/react-native-archipelago/releases)

# How To Play
* Generate a game locally and host or generate directly on MWGG
* Download and install the app (.apk only for now, Google Play soon, iOS later)
* Open the app and connect using your connection info
* The initial connection message from the server does not show up in the chat. If you see the chat screen, you are connected!
* Wait for the app to generate locations and start playing
