# Kindergarten 2

## Where is the options page?

The [player options page for this game](../player-options) contains all the options you need to configure and export a
config file.

## What does randomization do to this game?

* Shuffle Starting Money
* Monstermon Cards
* Outfits

## Notes
* You can play multiple games at once with only one installation, as the Archipelago save files will remain, yet be hidden, when connecting to a different multiworld.
  * The exception is an exact copy of the same multiworld (same multiword seed, and same slot seed), as they will conflict their save files.
  * Even then, the game state is almost entirely decided dynamically, from the server, so a conflicting save file should not cause problems
