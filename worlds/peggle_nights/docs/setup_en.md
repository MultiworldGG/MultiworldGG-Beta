# Peggle Nights Randomizer Setup Guide

## Prerequisites 

- If you do not use MWGG: Install the Peggle Nights APWorld as you would any other custom APWorld
- Have a multiworld generated and a room hosted

## How to play

- Open Peggle Nights
- Open the MultiworldGG Launcher. Find and open the Peggle Nights Client
- Connect to the multiworld room with the server credentials, e.g. multiworld.gg:50125
- You should see a message that the Peggle Nights process was found
- You are ready to play! Make sure to check out the Peggle Nights tab in the client
